## Dataset

* `ALL_DATASET_Configs_FULL.xlsx`: This file contains the versions and configuration items obtained via the KernJC method for the CVEs included in both the KernJC and KernelCTF datasets.
* `KernJC_Configs_FULL.xlsx`: This file contains the versions and configuration items obtained via the KernJC method for the CVEs included in the KernJC dataset.
* `CVE_IDENTIFY_CONFIG-KernelCTFdata.xlsx`: This file contains the set of configuration items identified for the KernelCTF dataset during the **Candidate Identification** stage of FCC. Note that non-existent or false configuration items have not been removed.
* `CVE_IDENTIFY_CONFIG-KJCdata.xlsx`: This file contains the set of configuration items identified for the KernJC dataset during the **Candidate Identification** stage of FCC. Note that non-existent or false configuration items have not been removed.
* `CVE_IDENTIFY-KernelCTFdata.xlsx`: This file records CVE information, CWE information, summaries of semantic clues, and summaries of evidence-side clues for the KernelCTF dataset during the **Candidate Identification** stage of FCC.
* `CVE_IDENTIFY-KJCdata.xlsx`: This file records CVE information, CWE information, summaries of semantic clues, and summaries of evidence-side clues for the KernJC dataset during the **Candidate Identification** stage of FCC.
* `KernelCTFdata_Configs_mini.xlsx`: This file contains the minimal configuration set derived from the KernelCTF documentation.
* `KernJCdata_Configs_mini.xlsx`: This file contains the minimal configuration set obtained by reproducing vulnerabilities for the CVEs included in the KernJC dataset.
